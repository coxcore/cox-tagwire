_variable-attribute

 a 태그의 'href'와 'title' 속성이 적용된다. 

 'custom' 은 표준 속성이 아니므로 적용되지 않는다. 
 비표준 속성이어도 미리 선언해두면 적용된다. 

 tail 함수는 연달아 쓸 수 있다. 


data 속성은 'data-변수명'으로 설정된다.
jQuery plugin 모드인 경우는 data('변수명') 로 설정되어 속성에 보이지 않는다.


'_변수명-data-이름' 형태로 설정하면, 'data-이름' 속성으로 설정된다.
jQuery plugin 모드인 경우는 data('이름') 로 설정된다.
